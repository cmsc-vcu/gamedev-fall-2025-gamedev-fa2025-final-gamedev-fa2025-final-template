# Elevator Wipeout - Sidecars and PBRs - Fall 2025

Elevator Wipeout is a save-the-world game where the player does all they can to prevent bad guys from destroying a space elevator.  If the bad guys win, the earth is destroyed in a spectacular, slow-motion extravaganza.

This repo contains our work on the four-person game development projects for GameDev.

## Files in the repository

./unity/ - this folder contains the Unity game files associated with this game.

./website/ - this folder contains files associated with the *gh-pages* created for this game.

./docs/ - Don't edit files in this folder directly.  Edit the source documents in the *./website* folder, then use `quarto` to render the documents to HTML and store them in the *./docs* folder.
